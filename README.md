# Counter-Current Shell-and-Tube Heat Exchanger Model

## 📌 Description
This project simulates a shell-and-tube heat exchanger operating in counter-current configuration using a thermo-hydraulic 1D model.

## ⚙️ Features
- Steady-state energy balance
- Counter-current flow configuration
- Temperature profile computation along the exchanger
- Parametric study (flow rate)

## 📊 Outputs
- Temperature evolution of hot and cold fluids
- Heat transfer rate
- Thermal performance analysis

## 🛠️ Tools
- Python (NumPy, Matplotlib)
- Jupyter Notebook

## 🎯 Objective
Industrial-oriented simulation for energy and aerospace thermal systems.
